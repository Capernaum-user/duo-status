미해결 질문

1. **실기기 시각 확인은 사용자 몫**: 팬/핀치줌/탭 제스처의 실제 손맛, 하단 시트 슬라이드 애니메이션, HUD 줄바꿈이
실제 폰 화면에서 겹치지 않는지는 `http://localhost:8895/`를 모바일 브라우저(또는 데스크톱 브라우저의 기기
시뮬레이터)로 직접 열어 확인이 필요하다. 헤드리스 환경이라 실제 터치스크린 렌더링/제스처를 눈으로 볼 수 없었다.
2. **`$('genAt')` 버그 제거는 ORDER 범위 밖 판단**: 검토 시 되돌리길 원하면 알려달라(단, 되돌리면 폴링마다
"연결 끊김" 배너 깜빡임이 재발한다).
3. **`fitScale()` 타겟 폭(640px)은 보드 타일 배치 근사치**: 실제 스테이션 좌표(plan~done 대각선 폭 약 532px +
토큰 카드 여백)로 역산한 근사값이며, 정확한 "보드 전체가 딱 맞게" 보정은 아니고 "대략 보이도록"(ORDER 표현)
수준이다.
